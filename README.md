# Java Assessments – Route Course

This repository contains the Java assessments and practice exercises we cover during the review sessions in the Route course.  
It includes solved examples, code snippets, and small projects to help reinforce Java concepts such as data types, control flow, OOP, and basic problem‑solving.

## How to use
- Clone the repository to your local machine.
- Open the relevant folder for each session’s assessment.
- Run and test the code to understand the logic and improve your skills.

## Structure
- `session-1/` – Basic syntax, variables, and data types.
- `session-2/` – Control flow and Operators.
- `session-3/` – OOP basics (classes, objects, inheritance).
- `session-4/` – Collections, exceptions, and file handling.
- `session-5/` – Encapsulation, Inheritance, Polymorphism.
- `session-6/` – Abstraction, Interfaces, Overloading, and Overriding.
- `session-7/` – String, StringBuilder.
- `session-8/` – Lists, Exceptions, Enum.
